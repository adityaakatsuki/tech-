# 🚀 Company Technology Analysis Platform

A Python-based web application that automates the collection and analysis of publicly available company information. The platform uses web scraping techniques to gather company data, identify technology stacks, and present actionable insights through an interactive Streamlit dashboard.

## ✨ Features

- 🔍 Search and analyze company information
- 🌐 Automated web scraping of publicly available data
- 💻 Technology stack identification
- 📊 Interactive Streamlit dashboard
- 🗄️ SQLite database for efficient data storage
- 🔐 Google OAuth authentication
- 📜 Search history management
- ⚡ Fast and user-friendly interface

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/company-tech-analysis.git
```

### Navigate to the project folder

```bash
cd company-tech-analysis
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
streamlit run streamlit_app.py
```

## 🔄 Project Workflow

1. User searches for a company.
2. The scraper collects publicly available company information.
3. Data is cleaned and processed.
4. Information is stored in the SQLite database.
5. The dashboard displays company insights and technology analysis.
